# Letter Count

Write a method, `letter_count` that takes a string and returns a hash mapping each letter to its frequency. Do not include spaces.

For example, `letter_count("moon")` should return:

```
{
  "m" => 1,
  "o" => 2,
  "n" => 1
}
```
